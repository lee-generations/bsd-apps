# SABnzbd
https://sabnzbd.org

### Command to fetch script
```
fetch https://raw.githubusercontent.com/tschettervictor/bsd-apps/main/sabnzbd/sabnzbd-install.sh
```

Don't forget to
```
chmod +x sabnzbd-install.sh
```
